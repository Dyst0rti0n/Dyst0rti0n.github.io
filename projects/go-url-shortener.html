URL Generation Algorithm


// generateShortURL creates a unique short URL code
func generateShortURL(longURL string) string {
    // Create a hash of the URL + timestamp for uniqueness
    hasher := md5.New()
    hasher.Write([]byte(longURL + time.Now().String()))
    hash := hex.EncodeToString(hasher.Sum(nil))

    // Take first 8 characters for the short URL
    return hash[:8]
}

URL Redirection Handler


// redirectHandler handles redirecting short URLs to their original destinations
func redirectHandler(w http.ResponseWriter, r *http.Request) {
    vars := mux.Vars(r)
    shortURL := vars["shortURL"]

    // Look up the original URL
    originalURL, found := urlStore.Get(shortURL)
    if !found {
        http.Error(w, "Short URL not found", http.StatusNotFound)
        return
    }

    // Track the click
    go trackClick(shortURL, r.Header.Get("User-Agent"), r.RemoteAddr)

    // Redirect to the original URL
    http.Redirect(w, r, originalURL, http.StatusFound)
}

Implementation Details

Architecture

The application follows a clean architecture pattern with clear separation of concerns:

  • Handlers: Process HTTP requests and responses
  • Services: Contain business logic for URL shortening
  • Storage: Abstract interface for different storage backends
  • Analytics: Click tracking and usage statistics

Persistence

The service supports multiple storage backends:

  • In-Memory: Fast but non-persistent storage for development
  • Redis: Persistent storage with expiration support
  • Pluggable Interface: Easy to add new storage providers

Performance Optimizations

  • Connection pooling for Redis operations
  • Goroutines for non-blocking analytics tracking
  • Caching of frequently accessed URLs
  • Efficient hash-based URL generation algorithm
